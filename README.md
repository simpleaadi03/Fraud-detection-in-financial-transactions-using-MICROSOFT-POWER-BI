This project addresses the growing complexity of fraudulent financial activities driven by the rapid expansion of digital banking, e-commerce, and instant payment platforms. Because traditional rule-based and reactive detection frameworks struggle to keep pace with modern fraud schemes, this solution turns to an intelligent, data-driven system built on business intelligence tools.  Methodology and Core Concepts:
Since real-world financial data is highly restricted due to user privacy and strict protection laws, the project leverages a comprehensively generated synthetic dataset. This data mimics real-world transactional behavior and includes critical parameters such as Transaction IDs, User IDs, transaction amounts, timestamps, merchant categories, and geospatial data. Data preprocessing is managed within Power BI using the Power Query Advanced Editor to clean, format, and prepare the dataset for descriptive statistical testing and outlier analysis. DAX (Data Analysis Expressions) is applied to formulate risk scores and behavioral metrics.  Key Insights and Results:
By converting raw complex data into an architecture of three distinct interactive dashboards, the system uncovers temporal, behavioral, and geographic patterns of high-risk activities:  Temporal Patterns: Analytical graphs identify distinct time-based vulnerabilities, noting a heavy concentration of suspicious behavior during late-night and early morning hours (specifically between 1 AM and 5 AM).  Behavioral & Device Analysis: Point-of-Sale (POS) systems exhibited the highest counts of high-risk activity. Furthermore, an even 50/50 risk distribution was tracked between "usual" and "unusual" transaction devices, highlighting the critical need to flag device abnormalities.  Geographic Mapping: Global tracking identifies explicit regional spikes across North America, Europe, and Australia, enabling location-specific interventions.  Ultimately, the interactive dashboards provide real-time reporting and predictive visibility, empowering financial institutions to transition from a reactive posture to pre-emptive fraud mitigation.
